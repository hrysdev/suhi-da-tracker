## 🚀 Getting Started

```bash
# パッケージのインストール（※pnpm推奨）
pnpm install

# 開発サーバを起動
pnpm dev
```

## 🤔 Usage

1. 画像のページにある「結果をXでポスト」をクリックすると、自動的にスコアやタイピング速度などの結果が拡張機能に保存されます。

2. 拡張機能のアイコンをクリックすると、保存された結果が可視化できるページに遷移します。
